## O que estou construindo agora

### 🤖 [ai-agent-workflows](https://github.com/IzaacBaptista/ai-agent-workflows)
Sistema de orquestração multi-agente para tarefas de engenharia de software. Os agentes operam em ciclo **Triage → Plan → Execute → Replan → Critique → Persist**, com registro via `agentRegistry.ts`, validação de respostas LLM com Zod e controle de comandos por allowlist.

Casos de uso: análise automática de issues, diagnóstico de bugs e criação de PRs no GitHub — tudo via CLI.

`TypeScript` `Node.js` `OpenAI` `Zod` `Octokit`

---

### 🧠 [GroundedOS Lab](https://github.com/IzaacBaptista/GroundedOS-Lab)
Plataforma open-source para construir, estudar e avaliar sistemas de AI fundamentados — RAG pipelines, orquestração multi-agente, observabilidade e camadas de segurança. Projetada como um **laboratório de engenharia de AI em produção**: expõe a mecânica interna de cada componente, de chunking e embeddings até guardrails e evals.

Inclui modos Dev e Lab para inspecionar chunks recuperados, uso de tokens, latência por estágio e A/B testing de prompts.

`TypeScript` `Python` `Next.js` `Node.js` `pgvector` `OpenTelemetry`
